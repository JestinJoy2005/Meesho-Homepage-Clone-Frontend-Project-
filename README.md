# Meesho Homepage Clone

This project is a frontend clone of the Meesho homepage, developed using HTML and CSS.  
It replicates the overall layout and visual appearance of the Meesho landing page for learning and practice purposes.

---

Project Description

The Meesho Homepage Clone is a static web application focused on recreating the user interface of the Meesho website.  
It contains a navigation bar, search functionality UI, category navigation links, promotional banner section, QR code display, and service highlights such as easy returns, cash on delivery, and lowest prices.

This project helps in understanding real-world e-commerce website layouts and improving frontend development skills.

---

Features

- Header navigation bar with Meesho logo
- Search bar with placeholder text and search icon
- Category navigation menu
- Promotional banner section
- QR code section for website access
- Service highlights (Easy Returns, Cash on Delivery, Lowest Prices)
- Clean and user-friendly UI design

---

Technologies Used

- HTML5 for webpage structure  
- CSS3 for styling and layout design  

---

Purpose of the Project

- Practice frontend web development
- Learn website layout structuring
- Improve CSS styling and positioning skills
- Build a beginner-friendly portfolio project
